# Rotary-spectrophotometer
Designing a scheme of spectrophotometer which uses a rotor with difractional grating instead of regular monochromator. 
# motivation
Researching properties of colored solutions is easy with spectrophotometers. That method allows to measure concentrations of chemicals simultaneously, without making chemical titration. That's why I decided to make one at home. Usually classic monochromator is used in spectrophotometers. It has a complicated structure which is practically unreal to make in home contitions. 
# research question
The main task is to make a properly functioning spectrophotometer at home. In order to make the construction easier I decided to use rotor with defractional grating.
